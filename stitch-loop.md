# Stitch Loop (Skill)

## Definición

**Stitch Loop** es una [[skills|skill]] que enseña a los agentes a construir sitios web de forma iterativa utilizando Google Stitch. Esta skill implementa un patrón de bucle autónomo, donde el agente genera, evalúa y refina diseños de UI en un ciclo continuo, pasando el "testigo" entre diferentes etapas o sub-agentes hasta lograr el resultado deseado.

## Características Clave

-   **Desarrollo Iterativo**: Guía al agente a través de un proceso de diseño y refinamiento paso a paso.
-   **Bucle Autónomo**: Permite al agente operar de forma independiente en el ciclo de diseño, tomando decisiones y ajustando el proceso.
-   **Patrón de "Testigo"**: Simula un flujo de trabajo donde la responsabilidad pasa entre diferentes etapas o sub-agentes (ej. generación, evaluación, edición).
-   **Optimización Continua**: Fomenta la mejora constante del diseño a través de la evaluación y el ajuste.

## Uso Potencial en el Proyecto

Un agente con la skill Stitch Loop podría ser utilizado en nuestro proyecto para:

-   **Diseño de Componentes Complejos**: Iterar sobre el diseño de componentes de UI complejos hasta lograr la perfección.
-   **Exploración de Variantes**: Generar y evaluar múltiples variantes de diseño para una sección de la aplicación.
-   **Refinamiento de Flujos de Usuario**: Diseñar y optimizar flujos de usuario completos, como el proceso de compra de tickets.
-   **Automatización de Diseño**: Automatizar partes del proceso de diseño, liberando a los diseñadores humanos para tareas más estratégicas.

## Integración con Agentes

Esta skill sería utilizada por agentes como `@ux-ui-developer` o `@frontend-developer` para guiar su proceso de diseño iterativo con Google Stitch.

```bash
@skill stitch-loop
```

Una vez invocada, la skill proporciona las directrices y el acceso a las funcionalidades de Stitch para el agente.
