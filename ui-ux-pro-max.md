# UI/UX Pro Max (Skill)

## Definición

**UI/UX Pro Max** es una [[skills|skill]] avanzada diseñada para agentes de IA, que proporciona inteligencia de diseño UI/UX para aplicaciones web y móviles. Esta skill encapsula un vasto conocimiento sobre estilos de diseño, paletas de colores, combinaciones de fuentes, pautas de UX y tipos de productos, permitiendo a los agentes generar y evaluar diseños de interfaz de usuario de alta calidad.

## Características Clave

-   **Amplia Biblioteca de Estilos**: Incluye más de 50 estilos de diseño (ej. glassmorphism, brutalism, neumorphism, bento grid, dark mode).
-   **Paletas de Colores y Fuentes**: Ofrece 161 paletas de colores y 57 combinaciones de fuentes predefinidas.
-   **Guías de UX**: Incorpora 99 pautas de experiencia de usuario para asegurar diseños intuitivos y eficientes.
-   **Tipos de Productos**: Conocimiento sobre 161 tipos de productos para adaptar el diseño al contexto específico.
-   **Integraciones**: Soporte para diferentes stacks tecnológicos (React, Next.js, Vue, Svelte, SwiftUI, React Native, Flutter, Tailwind, shadcn/ui).
-   **Acciones de Diseño**: Permite a los agentes planificar, construir, crear, diseñar, implementar, revisar, arreglar, mejorar, optimizar, realzar, refactorizar y verificar código UI/UX.

## Uso Potencial en el Proyecto

Un agente con la skill UI/UX Pro Max podría ser invaluable para nuestro proyecto de sistema de ticketera en las siguientes áreas:

-   **Generación de Prototipos**: Crear rápidamente prototipos de interfaces de usuario para nuevas funcionalidades.
-   **Evaluación de Diseño**: Analizar diseños existentes o propuestos para identificar mejoras en usabilidad, accesibilidad y estética.
-   **Asistencia en la Implementación**: Proporcionar sugerencias de código o clases de [[tailwind|Tailwind CSS]] para implementar diseños específicos.
-   **Refactorización de UI**: Sugerir mejoras en el código de la interfaz para alinearlo con mejores prácticas de diseño.
-   **Aseguramiento de Calidad (QA)**: Verificar que la UI implementada cumpla con las especificaciones de diseño y las pautas de UX.

## Integración con Agentes

Esta skill sería utilizada por agentes como `@ux-ui-developer` o `@frontend-developer` para potenciar sus capacidades de diseño e implementación de UI.

```bash
@skill ui-ux-pro-max
```

Una vez invocada, la skill inyectaría sus conocimientos y directrices en el contexto del agente, permitiéndole tomar decisiones de diseño más informadas.
