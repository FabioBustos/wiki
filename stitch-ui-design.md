# Stitch UI Design (Skill)

## Definición

**Stitch UI Design** es una [[skills|skill]] que proporciona orientación experta para la creación de prompts efectivos en Google Stitch, la herramienta de diseño UI asistida por IA de Google Labs. Esta skill ayuda a los agentes a formular prompts precisos y accionables que generen diseños de interfaz de usuario de alta calidad para aplicaciones web y móviles.

## Características Clave

-   **Guía de Prompts**: Ofrece directrices sobre cómo estructurar prompts para obtener los mejores resultados de Stitch.
-   **Palabras Clave de Diseño**: Sugiere palabras clave y frases específicas relacionadas con UI/UX para refinar las solicitudes de diseño.
-   **Contexto de Diseño**: Ayuda a incluir contexto relevante en los prompts, como el tipo de aplicación, la audiencia, el estilo deseado y las funcionalidades clave.
-   **Optimización de Resultados**: Enseña a los agentes a iterar y mejorar los prompts para lograr diseños que cumplan con los requisitos.

## Uso Potencial en el Proyecto

Un agente con la skill Stitch UI Design podría ser utilizado en nuestro proyecto para:

-   **Generación de Diseños Iniciales**: Crear prompts para generar los primeros borradores de interfaces de usuario para el frontend [[nextjs]].
-   **Refinamiento de Componentes**: Mejorar prompts para obtener variaciones específicas o ajustes en componentes existentes.
-   **Exploración de Ideas**: Utilizar prompts para explorar diferentes enfoques de diseño para una funcionalidad.
-   **Colaboración con Diseñadores**: Traducir las ideas de los diseñadores en prompts efectivos para Stitch.

## Integración con Agentes

Esta skill sería utilizada por agentes como `@ux-ui-developer` o `@frontend-developer` para interactuar con Google Stitch de manera más efectiva.

```bash
@skill stitch-ui-design
```

Una vez invocada, la skill proporciona las directrices y el acceso a las funcionalidades de Stitch para el agente.
