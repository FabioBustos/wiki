# Stitch Design Taste (Skill)

## Definición

**Stitch Design Taste** es una [[skills|skill]] especializada para agentes de IA, diseñada para generar archivos `DESIGN.md` que definen sistemas de diseño semánticos. Esta skill impone estándares de UI premium y anti-genéricos, enfocándose en una tipografía estricta, color calibrado, layouts asimétricos, micro-movimientos perpetuos y rendimiento acelerado por hardware.

## Características Clave

-   **Estándares de Diseño Premium**: Genera sistemas de diseño que evitan estéticas genéricas de IA.
-   **Tipografía Calibrada**: Define tipografías con atención al detalle y la legibilidad.
-   **Color Asimétrico**: Crea paletas de colores con un enfoque en la armonía y el impacto visual.
-   **Layouts Asimétricos**: Diseña estructuras de interfaz que son visualmente interesantes y dinámicas.
-   **Micro-movimientos**: Incorpora animaciones sutiles y fluidas para mejorar la [[experiencia-de-usuario|experiencia de usuario]].
-   **Rendimiento Acelerado por Hardware**: Considera la optimización para un rendimiento fluido de la UI.

## Uso Potencial en el Proyecto

Un agente con la skill Stitch Design Taste podría ser utilizado en nuestro proyecto para:

-   **Definición de Sistema de Diseño**: Crear la base de nuestro sistema de diseño, asegurando una estética única y de alta calidad para el frontend [[nextjs]].
-   **Guías de Estilo**: Generar guías de estilo detalladas para los componentes de UI.
-   **Revisión de Diseño**: Evaluar la coherencia y la calidad estética de los diseños propuestos.
-   **Optimización de UI**: Sugerir mejoras en el diseño para alinear con principios de rendimiento y micro-movimientos.

## Integración con Agentes

Esta skill sería utilizada por agentes como `@ux-ui-developer` o `@architect` para definir la dirección estética del proyecto.

```bash
@skill stitch-design-taste
```

Una vez invocada, la skill proporciona las directrices y el acceso a las funcionalidades de Stitch para el agente.
