# Stitch Design (Skill)

## Definición

**Stitch Design** es una [[skills|skill]] que proporciona un punto de entrada unificado para el trabajo de diseño con Google Stitch, una herramienta de diseño UI/UX asistida por IA. Esta skill permite a los agentes mejorar prompts de diseño, sintetizar sistemas de diseño y generar o editar pantallas de alta fidelidad, aprovechando las capacidades de Stitch MCP.

## Características Clave

-   **Mejora de Prompts**: Asiste en la creación de prompts de diseño más efectivos, incorporando palabras clave de UI/UX y elementos de atmósfera.
-   **Síntesis de Sistemas de Diseño**: Ayuda a generar archivos `DESIGN.md` que definen sistemas de diseño, aplicando estándares de calidad premium (tipografía, color, layouts asimétricos, micro-movimientos).
-   **Generación/Edición de Pantallas**: Permite a los agentes generar nuevas pantallas o editar existentes con alta fidelidad a través de Stitch MCP.

## Uso Potencial en el Proyecto

Un agente con la skill Stitch Design podría ser utilizado en nuestro proyecto para:

-   **Diseño Rápido de UI**: Generar rápidamente maquetas o prototipos de interfaces para el frontend [[nextjs]].
-   **Alineación con el Diseño**: Asegurar que la implementación de la UI se alinee con los principios de diseño definidos.
-   **Exploración de Estilos**: Experimentar con diferentes estilos visuales y estéticas para los componentes de la aplicación.
-   **Documentación de Diseño**: Ayudar a formalizar y documentar el sistema de diseño del proyecto.

## Integración con Agentes

Esta skill sería utilizada por agentes como `@ux-ui-developer` o `@architect` para interactuar con Google Stitch.

```bash
@skill stitch-design
```

Una vez invocada, la skill proporciona las directrices y el acceso a las funcionalidades de Stitch para el agente.
